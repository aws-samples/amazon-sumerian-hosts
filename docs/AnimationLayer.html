Class: AnimationLayer

AnimationLayer(optionsopt, blendModeopt, weightopt, transitionTimeopt, easingFnopt)

Class for managing a set of animations where only one state can be active at
any given time.

Constructor

new AnimationLayer(optionsopt, blendModeopt, weightopt, transitionTimeopt, easingFnopt)

Parameters:
Name Type Attributes Default Description
options Object <optional>

Options for the animation layer.

Properties
Name Type Description
name string

Name of the layer. Names must be unique to the
animation feature that contains the layer.
blendMode LayerBlendModes <optional>
 DefaultLayerBlendMode

Type of blending to use for all states controlled by the layer.
weight number <optional>
 1

The amount of influence the layer's current
animation has over the result for the host.
transitionTime number <optional>
 0

The default amount of time to use when
playing and resuming animations.
easingFn function <optional>

The default easing function to use when
transitioning between animations and setting layer weight.
